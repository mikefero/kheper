List all nodes connected to a host

Retrieve a list of all nodes connected to a specific host or address.

path Parameters
host
required
string
Example: example-host.com

The RFC 1123 IP address or hostname of the control plane connected to the service mesh. Must be between 1 and 253 characters long, and each segment separated by dots can contain alphanumeric characters and hyphens, but cannot start or end with a hyphen.

Retrieve a specific resource from a node payload

host
required
string

The RFC 1123 IP address or hostname of the control plane connected to the service mesh. Must be between 1 and 253 characters long, and each segment separated by dots can contain alphanumeric characters and hyphens, but cannot start or end with a hyphen.

node-id
required
string <uuid>
Example: 123e4567-e89b-12d3-a456-426614174000

The node ID

resource
required
string
Example: cpu

The resource name
